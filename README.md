## D-RoC Command Center

**NEW:** Use the CLI to navigate your universe:

```bash
cd droc-cli && npm install && npm link
droc status          # See all projects
droc find "echoes"   # Search anything
droc work "chalk"    # Load project context
droc list games      # List all games
```
