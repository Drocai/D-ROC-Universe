## 5. DATA STORAGE & SECURITY

### **5.1 Where We Store Data**

- **Primary:** Firebase (US servers)
- **Backup:** Automated daily backups
- **CDN:** Netlify global edge network

---

### **5.2 How Long We Keep Data**

**Active Accounts:**
- Retained while account is active
- Plus reasonable period for backups

**Deleted Accounts:**
- Personal data deleted within 30 days
- Some anonymized data retained for analytics
- Legal hold data retained as required

**Content:**
- Submitted tracks retained indefinitely (unless removed)
- Ratings retained for platform integrity
- Transaction history retained for auditing

---

### **5.3 Security Measures**

- HTTPS encryption (SSL/TLS)
- Password hashing (bcrypt)
- Firebase security rules
- Regular security audits
- Access controls and logging
- Automated threat detection

**However, no system is 100% secure. Use strong passwords and protect your account.**
